## Install

```bash
npm i -g @glrs-dev/cli
```

Requires [Bun](https://bun.sh) on PATH at runtime.

## Usage

```bash
glrs oc install            # → OpenCode harness install
glrs wt new my-feature     # create a worktree
glrs wt list               # list all worktrees
glrs wt switch             # interactive picker

glrs --help
glrs <subcommand> --help
```

## Subcommands

- `glrs oc <args>` — dispatches to [`@glrs-dev/harness-opencode`](/harness-opencode/) (bundled as a dependency).
- `glrs wt <args>` — worktree management, handled natively.

The `harness-opencode` bin remains available directly for power users who prefer the untagged entry point.

## What's not in here

- **`@glrs-dev/assume`** — the SSO credential manager is a standalone Rust binary. Install separately with `npm i -g @glrs-dev/assume`.
