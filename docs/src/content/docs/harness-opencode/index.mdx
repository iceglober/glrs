## What it is

An opinionated agent harness for [OpenCode](https://opencode.ai). Ships:

- **PRIME** — the primary agent with a five-phase workflow (intent → frame → plan → execute → verify → handoff)
- **Subagents** — `@plan`, `@qa-reviewer`, `@qa-thorough`, `@code-searcher`, `@explore`, `@lib-reader`, and more
- **Slash commands** — `/fresh`, `/ship`, `/review`, `/autopilot`, `/research`, `/init-deep`, `/costs`
- **Skills** — bundled domain skills (writing, research, QA methodology)
- **Pilot subsystem** — autonomous task execution driven by `pilot.yaml` DAGs
- **MCP wiring** — Serena, git, and more
