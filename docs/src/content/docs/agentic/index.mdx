## Install

```bash
npm i -g @glrs-dev/cli
```

Two equivalent bins ship with the package: `gs-agentic` and `gsag` (shorter). Use them via `glrs agentic`.

## What it is

A CLI for AI-native development workflows — worktree creation/management, plan state tracking, skill installation for Claude Code sessions.

## Slash commands (for Claude Code)

The CLI also installs slash commands into Claude Code:

```bash
glrs agentic skills
```

After install: `/work`, `/ship`, `/deep-plan`, `/research-web`, and more are available inside Claude Code sessions.
