Publish the background-jobs TUI sidebar as a real opencode plugin. Install with `opencode plugin @glrs-dev/opencode-background-sidebar`; it adds a "background jobs" section to the sidebar listing the current session's harness background jobs (`background_run`) with live status, title, and 2s refresh. Promoted from the `examples/` spike — now a published package, per-session filtered.
