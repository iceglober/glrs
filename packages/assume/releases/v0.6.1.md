# v0.6.1

Released: 2026-04-08

## Changes

### Fixed
- Fix daemon deadlock caused by holding write lock during desktop notification on session expiry
- Fix unsigned binary hanging on macOS when installed to directories with `com.apple.provenance` xattr
- Add 15-second timeouts to all credential fetch and session refresh operations in daemon
- Add 30-second timeout to HTTP connection handling
- Move `notify_session_expired` to `spawn_blocking` to prevent blocking the async runtime
- Move `load_active_context` fallback to `spawn_blocking` for non-blocking file I/O
- Increase credential endpoint validation timeout from 3s to 10s and retry delay from 2s to 3s
- Ad-hoc codesign binaries after auto-upgrade download to prevent macOS launch hang
