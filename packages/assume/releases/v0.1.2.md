# v0.1.2

Released: 2026-04-07

## Changes

### Fixed

- `use` now validates session tokens before allowing context switch — prevents switching to a context with no valid session
- `use` auto-launches `login` when no valid session exists instead of failing with an error
- `use` exports `AWS_CONTAINER_AUTHORIZATION_TOKEN` alongside the credential URI — fixes silent credential timeouts
- `use` validates the credential endpoint after switching and auto-restarts the daemon if unhealthy
- `exec` clears `AWS_CONTAINER_CREDENTIALS_FULL_URI` from child processes so AWS SDK uses injected static creds
- `exec` defaults to the active context when `--profile` is omitted
- Daemon startup kills all orphaned `gs-assume serve` processes, not just the PID file process
- `ensure_daemon_running` now health-checks the daemon port and restarts if unresponsive
- Ad-hoc codesign binaries after download on macOS — prevents silent hang on launch after upgrade
