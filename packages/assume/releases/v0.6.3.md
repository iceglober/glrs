# v0.6.3

Released: 2026-04-09

## Changes

### Fixed
- Fix misleading "credential endpoint not responding" error when SSO session is expired
- `gsa use` now distinguishes between daemon unreachable (restart) and expired session (auto-login)
- Remove pointless daemon restart when credential endpoint returns non-200 due to expired tokens
- `gsa use` auto-launches login flow when credential endpoint indicates session needs refresh
