# v0.4.1

Released: 2026-04-07

## Changes

### New
- `gsa login gcp` writes Application Default Credentials (ADC) file so gcloud, Pulumi, Terraform, and all Google Cloud SDKs authenticate automatically
- `gsa use gcp <project>` exports `CLOUDSDK_AUTH_ACCESS_TOKEN` so gcloud CLI works in the current shell without separate `gcloud auth login`
