# v0.5.0

Released: 2026-04-07

## Changes

### New
- `gsa exec` now supports GCP: injects `CLOUDSDK_AUTH_ACCESS_TOKEN`, `GOOGLE_CLOUD_PROJECT`, and `CLOUDSDK_CORE_PROJECT` into the subprocess
- `gsa exec --provider gcp --profile <name>` flag narrows context matching to a specific provider
- `gsa exec --profile gcp:novelist-app` syntax for inline provider:profile matching

### Changed
- `gsa exec` now reads contexts from cache first (instant) instead of always hitting live API
- AI agents can run `gsa exec --provider gcp --profile <name> -- gcloud projects list` without needing shell eval
