# v0.5.1

Released: 2026-04-07

## Changes

### Fixed
- `gsa agent exec` now supports GCP contexts (was hardcoded to AWS only)
- `gsa agent exec --provider gcp` flag narrows context matching to a specific provider
- `gsa agent exec --profile gcp:project` syntax for inline provider:profile matching
- Agent exec injects `CLOUDSDK_AUTH_ACCESS_TOKEN`, `GOOGLE_CLOUD_PROJECT`, and `CLOUDSDK_CORE_PROJECT` for GCP contexts
