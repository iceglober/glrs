# @glrs-dev/agentic

## 1.0.0

### Major Changes

- First release under the `@glrs-dev` npm scope. Renamed from `@glorious/agentic` (never published to npm).
- No functional changes in this release. Bin names `gs-agentic` and `gsag` are preserved — existing shell scripts and muscle memory keep working.
- Source moved from [`iceglober/glorious`](https://github.com/iceglober/glorious) (now archived) to [`iceglober/glrs/packages/agentic/`](https://github.com/iceglober/glrs/tree/main/packages/agentic). Full git history preserved via `git-filter-repo`.

### Install

```bash
npm i -g @glrs-dev/agentic
# or
bunx @glrs-dev/agentic <command>
```
