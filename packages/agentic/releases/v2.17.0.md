# v2.17.0

Released: 2026-04-13

## Changes

### New
- `state task note --id <id> --body "..."` — attach notes to tasks for persistent context
- `state task notes --id <id> [--json]` — list notes for a task
- `state task transition --ids <comma-list> --phase <phase>` — batch transition multiple tasks
- `state task update --id <id> --depends-on <comma-list>` — set/replace task dependencies
- `state plan sync --stdin` — atomically create epic + tasks from a line-based format
- `epicProgress()` — task count breakdown by status (done/cancelled/inProgress/blocked/ready)
- `progressBar()` — visual progress bar for terminal output
- `status --epic <id>` — filter status view to a single epic with progress bar

### Changed
- TASK_PREAMBLE and BUILD_PREAMBLE now document batch transitions, task notes, plan sync, and progress
- `/deep-plan` Step 7c offers atomic `plan sync` as preferred option over individual `add-task` calls
- `ship` phase now classified as in-progress in epic progress tracking
- Progress bar counts cancelled tasks as resolved (not as incomplete)
