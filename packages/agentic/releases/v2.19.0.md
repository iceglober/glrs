# v2.19.0

Released: 2026-04-14

## Changes

### New
- `HANDOFF_RULE` constant in preamble.ts — global constraint for skill-to-skill handoffs using Authority + Commitment persuasion
- `buildHandoffBlock()` helper function — generates standardized AskUserQuestion + dispatch table + constraint block
- `HandoffOption` interface exported from preamble.ts
- 5 regression tests in cross-refs.test.ts verifying handoff format consistency

### Changed
- All 5 handoff skills (deep-plan, build, build-loop, deep-review, quick-review) now use standardized dispatch tables instead of prose-style bullet points
- Dispatch format uses structured `| User selects | YOUR ACTION |` table with authority language ("Execute IMMEDIATELY", "MUST", "Do NOT")
- Each handoff includes a constraint block forbidding text output before the Skill tool call
- gs-quick-review now has explicit dispatch blocks for both SHIP IT and NEEDS FIXES verdicts (previously deferred to deep-review with a one-liner)
