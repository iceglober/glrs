# v2.13.0

Released: 2026-04-11

## Changes

### New
- Add `claimed_by` and `claimed_at` fields to tasks for cross-session visibility
- Parallel Claude Code sessions can now see who is working on each task
- `--unclaim` flag on `task update` to manually clear a stale claim

### Changed
- `task show`, `task list`, `status`, and `ready` display claim info when set
- Skill preambles (TASK_PREAMBLE, BUILD_PREAMBLE) instruct agents to check for existing claims before starting work
- Transitioning to `implement` sets `claimed_by` to the actor; terminal phases (`done`, `cancelled`) clear it
