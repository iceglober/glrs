# v2.2.0

Released: 2026-04-07

## Changes

### Changed

- Move `hooks` command under `wt` subcommand group (`gsag wt hooks` instead of `gsag hooks`)
- Make hook execution resilient: individual commands in hook scripts can fail without aborting the hook or crashing the CLI
- Use `fmt.ts` helpers (`info`, `warn`) for hook output instead of raw ANSI escapes

### Fixed

- Fix duplicate package name in release titles (e.g. "gs-assume v0.2.0" instead of "glorious-assume assume-v0.2.0")
