# v1.5.0

Released: 2026-04-03

### New
- `gs skills --prefix` organizes hyphenated skills into subdirectories (e.g. `product-manager` → `product/manager`, invoked as `/product:manager`)
- Collision detection: warns when existing non-glorious files would be overwritten, prompts to use `--prefix`
- Stale file cleanup via `.glorious-skills.json` manifest — old files removed automatically when skills change or layout switches between flat/prefixed
- Auto-upgrade: minor/patch updates download automatically on any command; major bumps still require `gs upgrade`
