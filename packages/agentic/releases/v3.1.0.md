# v3.1.0

Released: 2026-04-15

## Changes

### New
- Add `POST /api/task` endpoint to state server for creating tasks from the web dashboard
- Add `SubmitForm` component to the web dashboard — textarea + submit button for queuing work requests
- Add verify badge on tasks in `verify` phase ("awaiting review")
- Add `AUTO_PREAMBLE` to preamble.ts — autonomous mode directives (no user interaction, failure budget, token-efficient output)
- Add `/plan-loop` skill — autonomous planning agent that polls for `understand`-phase tasks, runs /research + /deep-plan
- Add `/auto-loop` skill — autonomous implementation agent that claims tasks, implements, reviews, QAs, and transitions to `verify` (human gate)

### Changed
- Keyboard shortcut handler now suppresses shortcuts when typing in textarea (SubmitForm support)
- `DashboardView` accepts `onRefresh` prop to trigger data refresh after task submission
- Skill count in `GS_SKILL_NAMES` increased from 12 to 14
- `/gs` skill table updated with `/plan-loop` and `/auto-loop` entries
