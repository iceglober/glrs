# v2.1.0

Released: 2026-04-05

## Changes

### New
- Global worktree registry at `~/.glorious/worktrees.json` — tracks all worktrees across repos
- `gs-agentic wt` (bare) opens an interactive picker to select and open a worktree shell
- `gs-agentic wt delete` (no name) opens interactive multi-select for bulk deletion
- `gs-agentic wt list` shows all worktrees across all repos, grouped by repository
- Arrow keys and j/k vim bindings for interactive pickers
- Registry auto-prunes stale entries (deleted worktree dirs) on load

### Changed
- `wt list`, `wt delete`, and `wt` now work from any directory (no git repo required)
- `wt delete` searches the global registry first, falls back to repo-local resolution
- `wt create`, `wt checkout`, and `ensureWorktree` automatically register worktrees
- `wt cleanup` deregisters worktrees after deletion
