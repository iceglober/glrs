# v2.11.0

Released: 2026-04-11

## Changes

### New
- Configurable `--prefix <string>` flag on `gsag skills` — default installs canonical short names (`/think`, `/work`, `/fix`, etc.); use `--prefix gs-` to restore legacy `/gs-*` names
- `BUILTIN_COLLISIONS` set warns at install time if any skill names collide with Claude Code built-in commands
- `buildCommands(prefix?)` function and `GS_SKILL_NAMES` registry for programmatic skill name management

### Changed
- Default installed skill names are now short canonical names (e.g. `/think` instead of `/gs-think`, `/deep-plan` instead of `/gs-deep-plan`)
- `/fix` skill rewritten with TDD methodology: reproduce bug with failing test, fix, verify (typecheck + test + build), structured report
- `/qa` skill rewritten with REVIEW_PREAMBLE, 5-level diff strategy, DB-backed finding storage, and SHIP IT / NEEDS FIXES verdict
- All skill cross-references updated to use canonical short names
- `--prefix` flag changed from boolean (glorious/ subdirectory) to optional string (custom prefix)
- Severity scale in QA aligned with review skills (CRITICAL/HIGH/MEDIUM/LOW)
