# v2.11.0

Released: 2026-04-11

## Changes

### New
- Add `--stdin` flag to `gsag state plan set` for piping plan content directly via heredoc
- TTY guard on `--stdin` prevents silent hangs when called interactively
- Buffer type safety handles both Buffer and string stdin chunks

### Changed
- gs-deep-plan skill saves plans directly to global store via `--stdin` instead of writing to `.claude/plans/` (eliminates duplicate storage)
- gs-build-loop removes `.claude/plans/` fallback scan — plans are always in the global store
- TASK_PREAMBLE updated to reference `--stdin` form

### Fixed
- Help text corrected: `state spec` references updated to `state plan` (spec subcommand was renamed in v2.8.0)
- gs-deep-plan red flags and frontmatter aligned with new global-store-only policy
