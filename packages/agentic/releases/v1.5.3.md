# v1.5.3

Released: 2026-04-03

## Changes

### Fixed
- `gs skills --prefix` now installs all skills under a single `glorious/` subdirectory (e.g. `work.md` → `glorious/work.md`, invoked as `/glorious:work`). Previously it split on the first hyphen, scattering skills across many unrelated subdirs (`product-manager` → `product/manager`).
