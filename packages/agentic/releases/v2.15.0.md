# v2.15.0

Released: 2026-04-12

## Changes

### New
- Add `gsag state web` command — opens a read-only browser dashboard showing all epics, tasks, plans, reviews, and claims
- React 19 + htm CDN-based UI with sidebar navigation, task detail panel, and plan viewer
- Multi-repo support via `--all` flag — view state across all repos/worktrees
- Add `state.auto-open` setting to control browser auto-open behavior
- Add `listEpics({ all })` and `listAllRepos()` to state API for cross-repo queries
- Add `/api/state?all=true` endpoint for multi-repo JSON data

### Fixed
- Fix path traversal vulnerability in `/api/plan/:id` endpoint (validate against `[a-zA-Z0-9_-]+`)
- Remove CORS `Access-Control-Allow-Origin: *` header from state server
- Replace `exec()` with `execFile()` in browser open command to prevent command injection
- Eliminate all XSS vectors by replacing innerHTML-based rendering with React DOM
- Fix epic-level review summary not being populated in API responses
- Fix plan fetch triggering for tasks without plans
