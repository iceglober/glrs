# v2.22.0

Released: 2026-04-14

## Changes

### New
- Add `resolveFeedback(id)` — archives plan feedback as `feedback-resolved-{timestamp}-{suffix}.md` instead of deleting it, retaining data for future RL use
- Add `listResolvedFeedback(id)` — returns sorted list of archived feedback filenames
- Add `exportEpicTrace(epicId)` — exports a complete JSON trace of an epic's lifecycle (metadata, plan versions, feedback history, tasks with transitions/notes, reviews)
- Add `gsag state epic export --id <id>` CLI command for RL data export
- Add `EpicTrace` interface to `state.ts`

### Changed
- Rename CLI command `clear-feedback` → `resolve-feedback` (breaking: scripts using the old name will error)
- Update deep-plan skill to reference `resolve-feedback` instead of `clear-feedback`
- Epic trace reviews are now scoped per-task (aggregated from each task's review summary) instead of repo-wide

### Fixed
- Resolved feedback filenames include a random suffix to prevent sub-millisecond timestamp collisions
- Increased test setTimeout from 5ms to 20ms for sorted-files assertions to reduce CI flakiness
