# v2.18.0

Released: 2026-04-13

## Changes

### Changed
- `gsag state web` now shows all repos by default; use `--local` to restrict to current repo (replaces `--all`)

### Fixed
- Fix server crash when running `gsag state web` outside a git repository
- Fix repo labels in web dashboard — split on `/` to match `normalizeRemoteUrl` output format (`github.com/org/repo`)
- Wrap HTTP request handler in try/catch to return 500 JSON instead of crashing the process
- Add `repoId` parameter to `deriveEpicPhase`, `loadEpic`, and `reviewSummary` so cross-repo queries work without git context
- Seed `selectedRepo` from first API response to prevent poll-driven repo switches
- Revert repo tab bar threshold to `> 1` (single-tab bar was useless)
- Update help text to document `--local` flag and new default behavior
