# v2.5.0

Released: 2026-04-08

## Changes

### New
- Interactive scope picker for `gsag skills` — choose project or user scope via TUI
- `--user` and `--project` flags for explicit scope selection
- Research skill (`/research`) — master research orchestrator routing to local/web/auto
- Research-local skill (`/research-local`) — parallel codebase research with Explore subagents
- Dockerized e2e test suite for skills install lifecycle (14 tests)
- Unit test suite for install-skills (82 tests across 7 describe blocks)

### Changed
- `/ship` skill now monitors PR CI checks after creation and auto-diagnoses failures
- Refactored install-skills handler into composable pure functions (resolveClaudeDir, computeInstallPlan, executeInstall, formatInstallResult)
- Manifest write is now atomic (write-to-temp-then-rename)
- Target label detection uses explicit scope field instead of fragile path comparison

### Fixed
- E2e repair test now asserts exact source content match instead of just "not corrupted"
