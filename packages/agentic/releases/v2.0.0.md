# v2.0.0

Released: 2026-04-03

## Breaking

- CLI binary renamed from `gs` to `gs-agentic` (alias `gsag`)
- Package renamed from `glorious` to `@glorious/agentic`
- Package moved from repo root to `packages/agentic/`
- Install URL changed: `packages/agentic/install.sh`

## New

- Bun workspace monorepo structure (`packages/agentic/`, `packages/assume/`)
- `@glorious/assume` (`gs-assume` / `gsa`) — Rust-based multi-cloud SSO credential manager with AWS Identity Center support, daemon architecture, fuzzy context matching, and interactive TUI picker
- `gs-assume upgrade` command with auto-update for minor/patch versions
- REST API fallback for update checks when `gh` CLI is unavailable
- GitHub Actions release workflow for `gs-assume` with cross-compilation (macOS + Linux, amd64 + arm64)

## Changed

- Root `package.json` is now a workspace root (no direct dependencies)
- Release workflow updated for monorepo paths
- `tsconfig.json` types field updated from `bun-types` to `@types/bun`
