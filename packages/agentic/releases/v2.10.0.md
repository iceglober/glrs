# v2.10.0

Released: 2026-04-11

## Changes

### New
- `gsag plan review --id <id>` — opens a plan in the browser with a clean rendered view and per-step inline feedback buttons
- `gsag state plan feedback --id <id>` — display plan feedback collected from browser review
- `gsag state plan clear-feedback --id <id>` — clear plan feedback after incorporating it
- Feedback storage module (`plan-feedback.ts`) — persists step-anchored feedback at `~/.glorious/plans/<repo>/<id>/feedback.md`
- HTML plan renderer (`plan-html.ts`) — converts plan markdown to a self-contained HTML page using `marked`
- Local HTTP server (`plan-server.ts`) — ephemeral server for plan review sessions (binds to 127.0.0.1)

### Changed
- `/gs-deep-plan` now checks for feedback when revising plans and clears it after incorporating
- `plansDir()` in `state.ts` is now exported for use by the feedback module

### Dependencies
- Added `marked` (^18.0.0) for markdown-to-HTML rendering
